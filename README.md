# Bayesian Uncertainty Modeling for Large-Scale Project Management

This repository contains the research workflow, code, experiments, and datasets for a doctoral-level project on modeling uncertainty in large-scale construction and infrastructure projects using Bayesian probabilistic methods.

The project focuses on developing a unified Bayesian framework that integrates:
- Bayesian time-series forecasting
- Hierarchical modeling for correlated project components
- Probabilistic costâ€“schedule risk propagation
- Scenario-based decision optimization under uncertainty

The goal is to create a robust, extensible, and real-world applicable modeling pipeline suitable for publication and funding proposals.

---

## 1. Objectives

### Primary Objective
Develop a Bayesian probabilistic model capable of forecasting cost and schedule behavior in complex multi-phase construction projects, with accurate uncertainty quantification.

### Secondary Objectives
- Analyze real-world megaproject datasets and identify structural sources of uncertainty.
- Compare Bayesian modeling (PyMC / PyStan) with classical Monte Carlo simulation.
- Build a synthetic dataset generator for benchmarking models.
- Provide reproducible notebooks, experiments, and documentation.
